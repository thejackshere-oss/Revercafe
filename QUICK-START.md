## ⚡ 5-Minute Setup

### Step 1: Download Files (1 minute)
You have received these files:
- ✅ `index.html` - Main website file
- ✅ `config.js` - Configuration (optional)
- ✅ `README.md` - Full documentation
- ✅ `CUSTOMIZATION-GUIDE.md` - How to customize
- ✅ Plus 5 other guides

### Step 2: Choose Hosting (1 minute)

**Option A: GitHub Pages (FREE, RECOMMENDED)**
1. Create account at https://github.com (if you don't have one)
2. Go to https://github.com/new
3. Create new repository named: `revercafekolkata`
4. Click "Create repository"

**Option B: Free Alternatives**
- **Netlify**: https://app.netlify.com (connects to GitHub)
- **Vercel**: https://vercel.com (connects to GitHub)
- **Traditional Hosting**: Upload via FTP to your provider

### Step 3: Upload Files (1.5 minutes)

**On GitHub**:
1. In your new repository, click "Add file" → "Upload files"
2. Drag & drop `index.html` into the upload area
3. Click "Commit changes"

**That's it!** 🎉

### Step 4: Enable Publishing (1 minute)

1. Go to Settings → Pages
2. Under "Source", select: `main` branch
3. Under "Branch", select: `/ (root)`
4. Click "Save"
5. Wait 2-3 minutes

Your website is now LIVE! ✅

### Step 5: Find Your URL (0.5 minutes)

Your website will be at:
```
https://YOUR_GITHUB_USERNAME.github.io/revercafekolkata
```

Share this link everywhere! 📱

## 🆘 Common Issues & Fixes

### Website shows 404 error
- Check spelling of repository name
- Ensure index.html is in root folder
- Wait 5 minutes for GitHub Pages to process

### Images don't load
- Ensure image files uploaded to same directory
- Check filenames match exactly (case-sensitive)
- Wait for GitHub to sync

### Links don't work
- Check URL spelling
- Ensure links start with https://
- Test in incognito mode

### Mobile menu doesn't work
- Clear browser cache
- Close and reopen browser
- Test in different browser

### Form doesn't submit
- Check WhatsApp number is correct (919XXXXXXXXXX format)
- Test WhatsApp link directly
- Ensure JavaScript is enabled
